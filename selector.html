ELEMENT SELECTOR

The CSS element selector, also known as the type selector, is used to select and apply styles to all HTML elements of a specific type. It targets elements based on their HTML tag name.

CLASS SELECTOR

A CSS class selector is used to apply styles to one or more HTML elements that share a specific class name. This selector provides a flexible way to group elements and apply consistent styling across different parts of a webpage, regardless of their HTML tag type.

UNIVERSAL SELECTOR

The universal selector in CSS is a powerful tool used to select all elements within an HTML document. It is represented by an asterisk (*).
